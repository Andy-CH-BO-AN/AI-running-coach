# Documentation and PM Workflow (Antigravity Adapter)

This workflow is for updating documentation, READMEs, and project management artifacts.

## Prerequisites
1. Read the default dev workflow: [`ai/shared/instructions.md`](../../ai/shared/instructions.md)
2. Activate communication overlay: [`ai/skills/token-decrease/SKILL.md`](../../ai/skills/token-decrease/SKILL.md)

## Steps
1. **Update Docs**: Make necessary changes to the documentation files.
2. **Review**: Evaluate the changes using the README/PM review skill: [`ai/skills/readme-pm-review/SKILL.md`](../../ai/skills/readme-pm-review/SKILL.md).
3. **Commit**: Apply git changes according to conventions: [`ai/skills/git-change-conventions/SKILL.md`](../../ai/skills/git-change-conventions/SKILL.md).
