# UI/UX Development Workflow (Antigravity Adapter)

This workflow guides frontend and dashboard UI/UX development.

## Prerequisites
1. Read the default dev workflow: [`ai/shared/instructions.md`](../../ai/shared/instructions.md)
2. Activate communication overlay: [`ai/skills/token-decrease/SKILL.md`](../../ai/skills/token-decrease/SKILL.md)

## Steps
1. **Develop**: Implement UI changes. Keep the frontend stack to Vanilla HTML/JS/CSS without any build step, as requested by the project constraints.
2. **UI/UX Audit**: Review and test the UI, layouts, and usability using the UI/UX agent instructions: [`ai/shared/uiux.agent.md`](../../ai/shared/uiux.agent.md).
3. **Commit**: Apply git changes according to conventions: [`ai/skills/git-change-conventions/SKILL.md`](../../ai/skills/git-change-conventions/SKILL.md).
